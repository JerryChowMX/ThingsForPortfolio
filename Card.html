HTML 5

HTML 5 is the latest version of Hypertext Markup Language, the code that describes web pages.

It's a core technology markup language of the Internet used for structuring and presenting content on the World Wide Web.

It introduces many new features and capabilities that enable modern, rich, interactive websites.

CSS 3

Cascading Style Sheets is a stylesheet language used to describe the presentation of a document written in HTML

It is a programing language that enables the styling of web pages, including layout, colors, and fonts.

CSS provides more flexibility and control in the specification of presentation characteristics.

JavaScript

JavaScript is primarily used for creating interactive and dynamic content on websites, because it enables the implementation of complex features on web pages.

This programming language is an essential part of modern web development, both on the client-side and, more recently, on the server-side using environments such as Node.js.
